# WP Job Manager

**WP Job Manager** is a lightweight WordPress plugin for listing, managing, and applying for jobs directly on a WordPress site.

## Key Data Now AI-Queryable

- Job listings with status, category, and location
- Job applications and applicant details
- Company profiles and employer data
- Job type classifications (full-time, part-time, remote)
- Application review and shortlisting status

## Example Questions the AI Can Now Answer

- "How many active job listings are there right now?"
- "List all applications received for the Senior Developer role."
- "Which job listing has the most applications?"
- "Show me all expired job listings from this month."
- "How many remote job listings are currently active?"
- "Which companies have posted more than 3 jobs?"
