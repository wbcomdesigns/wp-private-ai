# AffiliateWP

**AffiliateWP** is a premium affiliate management plugin for WordPress that tracks referrals, manages affiliates, processes payouts, and provides affiliate marketing tools.

> Note: AffiliateWP is a commercial plugin not available on wordpress.org. This adapter was written manually using their public REST API documentation, demonstrating the concept works for paid plugins too.

## Key Data Now AI-Queryable

- Affiliate roster, status, and performance stats
- Referral records and commission amounts
- Payout history and payment status
- Affiliate creative assets (banners and links)
- Affiliate registration and approval queue
